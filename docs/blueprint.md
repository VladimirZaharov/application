# **App Name**: PropoCraft

## Core Features:

- Динамическое заполнение шаблонов: Автоматически вставлять описания выбранных проблем в предопределенные шаблоны для быстрого создания персонализированных предложений.
- Настройка тона с помощью ИИ: Использовать инструмент генеративного ИИ для настройки тона текста предложения в зависимости от выбранного профиля клиента (например, формальный, неформальный, технический).
- Интерактивная библиотека проблем: Просматривайте и выбирайте из каталогизированной библиотеки общих проблем и решений, относящихся к различным отраслям.
- Предварительный просмотр в реальном времени: Отображать предварительный просмотр предложения в реальном времени по мере его создания, позволяя пользователям мгновенно видеть изменения.
- Настраиваемый брендинг: Включить логотипы компаний и цветовые схемы в предложение для профессионального и фирменного вида.
- Экспорт PDF: Загрузите сгенерированное предложение в виде высококачественного PDF-файла для удобного обмена и распространения.

## Style Guidelines:

- Основной цвет: темно-синий сапфир (#2C3E50) для доверия и профессионализма.
- Цвет фона: светло-серый (#F0F2F5) для обеспечения удобочитаемости и чистого макета.
- Акцентный цвет: приглушенный фиолетовый (#8E44AD) для выделения ключевых элементов и призывов к действию.
- Основной текст: 'PT Sans', шрифт без засечек для баланса современности и тепла. Текст заголовка: 'PT Sans', как и основной.
- Используйте простые геометрические значки для представления различных разделов и функций, сохраняя чистую и профессиональную эстетику.
- Реализуйте чистый макет на основе сетки с большим количеством пустого пространства для улучшения читабельности и визуальной привлекательности.
- Используйте тонкие эффекты перехода для интерактивных элементов, чтобы улучшить взаимодействие с пользователем, не отвлекая.
